if you're a beginner, follow the steps below
1- download python - https://www.python.org/downloads make sure to get the latest version
2- configure "setup.ini" how you'd like (enter tokens and webhook)
3- open the "USERNAME GENERATOR" in cmd
4- install the required files by "pip install -r requirements.txt"
5- run the script by using the command "python main.py"

ENJOY
